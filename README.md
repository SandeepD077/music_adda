# 🎵 Spotify Clone

This is a simple **Spotify Clone** built using **HTML, CSS, and JavaScript**. It mimics the basic functionality of an audio player with a list of songs, play/pause controls, progress bar, and next/previous buttons.

---

## 📚 Project Overview

The project includes the following features:
- 🎧 **Play/Pause Songs:** Click on any song or the play button to start playing.
- ⏭️ **Next/Previous Buttons:** Switch between songs easily.
- 📶 **Progress Bar:** Seek through the song by adjusting the progress bar.
- 📸 **Dynamic Song Covers:** Each song updates its cover and name dynamically.
- 🎥 **GIF Animation:** Shows a playing animation when a song is active.

---

## 📂 Folder Structure

🎨 Technologies Used
HTML5: Structure of the application.

CSS3: Styling and responsiveness.

JavaScript: Handles the music player logic.

📄 Files Description
1. index.html
Contains the structure of the page.

Displays the song list and controls.

2. style.css
Provides styling for the player.

Includes media queries for responsive design.

3. script.js
Handles play/pause, song switching, and progress updates.

Updates UI dynamically for active songs.

🎧 Features in Detail
Dynamic Song List: Automatically updates the song list from the songs array.

Play/Pause Control: Click to toggle between play and pause.

Next/Previous Controls: Skip to the next or previous song.

Progress Bar: Allows seeking through the song.

Playing Animation: GIF animation while the song is playing.

🛠️ How to Customize
Add new songs in the songs/ folder.

Add corresponding cover images in the covers/ folder.

Update the songs array in script.js with new songs and paths.
